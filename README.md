# Deep-learning-Notes
### [SOI] 🧠 深度学习学习计划
本项目记录学习李沐老师《动手学深度学习》的历程，目标通过寒假30天时间快速掌握深度学习核心基础与PyTorch实操技能，从张量操作到模型搭建，循序渐进夯实入门功底。

#### 📅 第一阶段：PyTorch张量核心操作 (Day 1)
重点掌握深度学习最基础的数据结构——张量（Tensor）的核心操作，理解"按元素运算"等核心思想：
- 张量创建：arange生成连续序列，指定float32数据类型适配深度学习场景；
- 形状重塑：reshape调整张量维度（如3×4矩阵），不改变数据本身；
- 张量拼接：cat函数按dim=0（行）/dim=1（列）维度拼接，明确非拼接维度形状一致规则；
- 按元素运算：exp函数实现自然指数运算，理解逐元素独立计算的特性。
